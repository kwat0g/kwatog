# Track 2, Task 3: Edge Machine Condition Ingest (IoT) — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development. Steps use `- [ ]` syntax.

**Goal:** Add `POST /edge/v1/condition` so an IoT sensor (token ability `edge:condition`) bound to a machine can stream metric readings (temperature, vibration, pressure, current, oil_quality). Reuses `PredictiveMaintenanceService::recordAndEvaluate()` so threshold breaches still auto-create corrective MWOs after N consecutive breaches.

**Architecture:**

```
IoT sensor (machine-bound) → POST /edge/v1/condition { metric, value, unit?, recorded_at?, notes?, idempotency_key? }
  ↓
EdgeConditionController (ability:edge:condition)
  ↓
EdgeConditionIngestService::ingest(EdgeDevice $device, array $payload, ?string $idemKey)
  ├─ require device.machine_id → 422 device_not_bound_to_machine
  ├─ resolve edge-system user (cached via EdgeSystemUserResolver, extracted from T2.2)
  ├─ idempotency cache lookup → return cached reading if hit
  └─ delegate to PredictiveMaintenanceService::recordAndEvaluate([
         'machine_id'  => $device->machine_id,
         'metric'      => $payload.metric,
         'value'       => $payload.value,
         'unit'        => $payload.unit ?? <enum default>,
         'recorded_at' => $payload.recorded_at ?? now,
         'source'      => 'iot_sensor',
         'notes'       => $payload.notes,
       ], $systemUser)
  ↓
Standard predictive flow: reading row + breach check + consecutive-breach gate + MWO create + log.
```

Key reuse:
- `PredictiveMaintenanceService::recordAndEvaluate()` already returns `['reading', 'triggered', 'reason?', 'work_order?']` — controller forwards.
- `Auth::shouldUse('web') + Auth::onceUsingId($systemUserId)` wrapper from T2.2's `EdgeOutputIngestService` is **factored out** into `EdgeSystemUserResolver` (Common service inside Edge module) so T2.3 doesn't duplicate it.

Response on breach + MWO trigger:

```jsonc
{
  "data": {
    "reading_id": "yR3kLm",
    "metric": "temperature",
    "value":  "112.500",
    "unit":   "celsius",
    "triggered": true,
    "reason":   "temperature reading 112.500 celsius exceeds safe threshold.",
    "work_order": {
      "id":         "qZ8nKx",
      "mwo_number": "MWO-202606-0014",
      "machine_id": "wK5mPx"
    }
  }
}
```

Response on no breach:

```jsonc
{
  "data": {
    "reading_id": "yR3kLm",
    "metric": "vibration",
    "value":  "1.200",
    "unit":   "mm/s",
    "triggered": false
  }
}
```

Idempotency: same `idempotency_key` within 24h → cached reading hash id returned, no new row.

## Background

- `PredictiveMaintenanceService::recordAndEvaluate(array $data, User $by): array` — canonical entry point. Already handles defaults, breach check, consecutive-breach gate (currently 3), MWO creation, audit logging. **DO NOT REINVENT.**
- Valid metrics: `temperature | vibration | pressure | current | oil_quality`. Default units per metric live in `PredictiveMaintenanceService::defaultUnit()`.
- `MachineConditionReading.source` accepts `manual | iot_sensor | plc | api` (varchar(50)). Edge ingest uses `iot_sensor`.
- `recorded_by` is nullable FK; we still pass the cached edge-system user for audit clarity.
- T2.2 already established the `Auth::shouldUse('web') + onceUsingId(...)` pattern so `HasAuditLog::Auth::id()` doesn't return the EdgeDevice PK. Extract that into `EdgeSystemUserResolver`.
- `ability:edge:condition` middleware alias works because the Sanctum `ability` middleware was registered globally in T2.1.
- HashIDs: response includes `reading_id` and `work_order.id` as hash. Input has no hash IDs (device owns binding, no body IDs).
- Idempotency cache key: `edge:condition:idem:{key}` → `reading_id` (int). 24h TTL same as T2.2.

```php
<?php

declare(strict_types=1);

namespace App\Modules\Edge\Services;

use App\Modules\Auth\Models\Role;
use App\Modules\Auth\Models\User;
use Illuminate\Support\Facades\Auth;
use Illuminate\Support\Facades\Cache;
use Illuminate\Support\Str;

/**
 * T2.2 / T2.3 shared. Resolves the audit-trail user that every Edge ingest
 * persists under (`recorded_by` on MachineConditionReading / WorkOrderOutput).
 *
 * Why this class exists: under `auth:edge_device`, `Auth::id()` returns the
 * EdgeDevice PK. Models that write audit rows (HasAuditLog) would then
 * violate `audit_logs.user_id → users(id)` FK. We pin the auth context to
 * a real `users` row for the duration of the call.
 */
class EdgeSystemUserResolver
{
    public const SYSTEM_USER_EMAIL = 'edge-system@ogami.internal';
    public const CACHE_KEY         = 'edge:system_user_id';

    public function id(): int
    {
        $cached = Cache::get(self::CACHE_KEY);
        if (is_int($cached) && User::query()->whereKey($cached)->exists()) {
            return $cached;
        }

        $user = User::query()->where('email', self::SYSTEM_USER_EMAIL)->first();
        if (! $user) {
            $roleId = Role::query()->where('slug', 'employee')->value('id')
                ?? Role::query()->orderBy('id')->value('id');
            $user = User::create([
                'name'                => 'Edge System',
                'email'               => self::SYSTEM_USER_EMAIL,
                'password'            => bcrypt(Str::random(40)),
                'role_id'             => $roleId,
                'is_active'           => true,
                'password_changed_at' => now(),
            ]);
        }
        Cache::forever(self::CACHE_KEY, $user->id);
        return $user->id;
    }

    /**
     * Run $fn impersonating the edge-system user on the `web` guard so
     * HasAuditLog's Auth::id() returns a valid users.id.
     */
    public function impersonate(callable $fn): mixed
    {
        $id = $this->id();
        $previous = Auth::getDefaultDriver();
        Auth::shouldUse('web');
        try {
            return Auth::onceUsingId($id) ? $fn() : $fn();
        } finally {
            Auth::shouldUse($previous);
        }
    }

    public function user(): User
    {
        return User::query()->whereKey($this->id())->firstOrFail();
    }
}
```

- [ ] Step 1:

```php
<?php

declare(strict_types=1);

namespace App\Modules\Edge\Services;

use App\Modules\Edge\Models\EdgeDevice;
use App\Modules\Maintenance\Models\MachineConditionReading;
use App\Modules\Maintenance\Services\PredictiveMaintenanceService;
use Illuminate\Support\Facades\Cache;
use Illuminate\Validation\ValidationException;

/**
 * T2.3 — Edge → machine condition reading ingest.
 *
 * Persists one reading per call and delegates breach detection +
 * corrective-WO triggering to PredictiveMaintenanceService.
 *
 * @return array{reading: MachineConditionReading, triggered: bool, reason?: string, work_order?: object}
 */
class EdgeConditionIngestService
{
    private const IDEMPOTENCY_TTL_SECONDS = 86400;

    public function __construct(
        private readonly PredictiveMaintenanceService $predictive,
        private readonly EdgeSystemUserResolver $systemUser,
    ) {}

    public function ingest(EdgeDevice $device, array $payload, ?string $idemKey = null): array
    {
        if (! $device->machine_id) {
            throw ValidationException::withMessages([
                'device' => ['device_not_bound_to_machine'],
            ]);
        }

        if ($idemKey) {
            $cached = Cache::get($this->cacheKey($idemKey));
            if (is_int($cached)) {
                $cachedReading = MachineConditionReading::find($cached);
                if ($cachedReading) {
                    return ['reading' => $cachedReading, 'triggered' => false, 'replayed' => true];
                }
            }
        }

        $data = [
            'machine_id'  => (int) $device->machine_id,
            'metric'      => (string) $payload['metric'],
            'value'       => (float) $payload['value'],
            'unit'        => $payload['unit'] ?? null,
            'recorded_at' => $payload['recorded_at'] ?? now(),
            'source'      => 'iot_sensor',
            'notes'       => $payload['notes'] ?? null,
        ];

        $result = $this->systemUser->impersonate(
            fn () => $this->predictive->recordAndEvaluate($data, $this->systemUser->user()),
        );

        if ($idemKey && isset($result['reading'])) {
            Cache::put($this->cacheKey($idemKey), (int) $result['reading']->id, self::IDEMPOTENCY_TTL_SECONDS);
        }

        return $result;
    }

    private function cacheKey(string $key): string
    {
        return 'edge:condition:idem:'.$key;
    }
}
```

NOTE: `PredictiveMaintenanceService::recordAndEvaluate()` lets `unit` default via its `defaultUnit($metric)`. Passing `null` unit is fine — it picks the default. Confirm by reading the service body before relying on that.

### Task D: Controller

**Files:** Create `api/app/Modules/Edge/Controllers/EdgeConditionController.php`

- [ ] Step 1:

```php
<?php

declare(strict_types=1);

namespace App\Modules\Edge\Controllers;

use App\Modules\Edge\Models\EdgeDevice;
use App\Modules\Edge\Requests\ConditionIngestRequest;
use App\Modules\Edge\Services\EdgeConditionIngestService;
use Illuminate\Http\JsonResponse;

class EdgeConditionController
{
    public function __construct(private readonly EdgeConditionIngestService $service) {}

    public function ingest(ConditionIngestRequest $request): JsonResponse
    {
        /** @var EdgeDevice $device */
        $device = $request->user();

        $payload = $request->validated();
        $idem = $payload['idempotency_key'] ?? $request->header('X-Idempotency-Key');
        unset($payload['idempotency_key']);

        $result = $this->service->ingest($device, $payload, $idem);
        $reading = $result['reading'];

        $body = [
            'reading_id' => $reading->hash_id,
            'metric'     => $reading->metric,
            'value'      => (string) $reading->value,
            'unit'       => $reading->unit,
            'triggered'  => (bool) ($result['triggered'] ?? false),
        ];
        if (! empty($result['reason']))    $body['reason']    = $result['reason'];
        if (! empty($result['replayed']))  $body['replayed']  = true;
        if (! empty($result['work_order'])) {
            $wo = $result['work_order'];
            $body['work_order'] = [
                'id'         => $wo->hash_id,
                'mwo_number' => $wo->mwo_number,
                'machine_id' => app('hashids')->encode($wo->machine_id),
            ];
        }

        return response()->json(['data' => $body], 201);
    }
}
```

<?php

declare(strict_types=1);

namespace Tests\Feature\Edge;

use App\Modules\Auth\Models\Role;
use App\Modules\Edge\Models\EdgeDevice;
use App\Modules\Maintenance\Models\MachineConditionReading;
use App\Modules\MRP\Models\Machine;
use Illuminate\Foundation\Testing\RefreshDatabase;
use Illuminate\Support\Facades\Cache;
use Tests\TestCase;

class EdgeConditionIngestTest extends TestCase
{
    use RefreshDatabase;

    protected function setUp(): void
    {
        parent::setUp();
        if (! Role::query()->where('slug', 'employee')->exists()) {
            Role::create([
                'name' => 'Employee', 'slug' => 'employee',
                'description' => 'Default test role', 'is_system' => false,
            ]);
        }
        Cache::flush();
    }

    private function iotDevice(?Machine $machine = null): EdgeDevice
    {
        return EdgeDevice::create([
            'serial_number' => 'IOT-'.uniqid(),
            'name'          => 'Sensor',
            'device_type'   => 'iot_sensor',
            'location'      => 'Press 1',
            'machine_id'    => $machine?->id,
        ]);
    }

    private function tokenFor(EdgeDevice $d, array $abilities = ['edge:condition']): string
    {
        return $d->createToken('t', $abilities)->plainTextToken;
    }

    private function machine(): Machine
    {
        return Machine::factory()->create();
    }

    private function post(string $token, array $body, array $headers = []): \Illuminate\Testing\TestResponse
    {
        return $this->withHeaders(array_merge([
            'Authorization' => "Bearer {$token}",
        ], $headers))->postJson('/api/v1/edge/v1/condition', $body);
    }

    public function test_normal_reading_persists_no_trigger(): void
    {
        $m = $this->machine();
        $r = $this->post($this->tokenFor($this->iotDevice($m)), [
            'metric' => 'vibration', 'value' => 1.2, 'unit' => 'mm/s',
        ]);
        $r->assertStatus(201);
        $r->assertJsonPath('data.triggered', false);
        $this->assertSame(1, MachineConditionReading::query()->where('machine_id', $m->id)->count());
    }

    public function test_single_breach_does_not_yet_trigger_mwo(): void
    {
        $m = $this->machine();
        $r = $this->post($this->tokenFor($this->iotDevice($m)), [
            'metric' => 'temperature', 'value' => 999, // way above default threshold
        ]);
        $r->assertStatus(201);
        $r->assertJsonPath('data.triggered', false);
        $this->assertStringContainsString('exceeds safe threshold', $r->json('data.reason') ?? '');
    }

    public function test_consecutive_breaches_create_mwo(): void
    {
        $m = $this->machine();
        $device = $this->iotDevice($m);
        $tok = $this->tokenFor($device);

        // Seed prior breaching readings as the system would have.
        $this->post($tok, ['metric' => 'temperature', 'value' => 999]);
        $this->post($tok, ['metric' => 'temperature', 'value' => 999]);
        $r = $this->post($tok, ['metric' => 'temperature', 'value' => 999]);

        $r->assertStatus(201);
        $r->assertJsonPath('data.triggered', true);
        $this->assertNotEmpty($r->json('data.work_order.mwo_number'));
    }

    public function test_unknown_metric_is_422(): void
    {
        $m = $this->machine();
        $r = $this->post($this->tokenFor($this->iotDevice($m)), [
            'metric' => 'humidity', 'value' => 50,
        ]);
        $r->assertStatus(422);
    }

    public function test_non_numeric_value_is_422(): void
    {
        $m = $this->machine();
        $r = $this->post($this->tokenFor($this->iotDevice($m)), [
            'metric' => 'temperature', 'value' => 'hot',
        ]);
        $r->assertStatus(422);
    }

    public function test_device_not_bound_is_422(): void
    {
        $r = $this->post($this->tokenFor($this->iotDevice(null)), [
            'metric' => 'temperature', 'value' => 50,
        ]);
        $r->assertStatus(422);
        $this->assertStringContainsString('device_not_bound_to_machine', $r->getContent());
    }

    public function test_idempotent_replay(): void
    {
        $m = $this->machine();
        $tok = $this->tokenFor($this->iotDevice($m));
        $first  = $this->post($tok, ['metric' => 'vibration', 'value' => 0.5, 'idempotency_key' => 'iot-001']);
        $second = $this->post($tok, ['metric' => 'vibration', 'value' => 0.5, 'idempotency_key' => 'iot-001']);

        $first->assertStatus(201);
        $second->assertStatus(201);
        $this->assertSame($first->json('data.reading_id'), $second->json('data.reading_id'));
        $this->assertSame(1, MachineConditionReading::query()->where('machine_id', $m->id)->count());
    }

    public function test_plc_token_cannot_post_condition(): void
    {
        $m = $this->machine();
        $d = $this->iotDevice($m);
        $tok = $d->createToken('p', ['edge:output'])->plainTextToken;
        $r = $this->post($tok, ['metric' => 'temperature', 'value' => 50]);
        $r->assertStatus(403);
    }

    public function test_no_token_is_401(): void
    {
        $this->postJson('/api/v1/edge/v1/condition', ['metric' => 'temperature', 'value' => 50])
            ->assertStatus(401);
    }

    public function test_unit_default_applied_when_omitted(): void
    {
        $m = $this->machine();
        $r = $this->post($this->tokenFor($this->iotDevice($m)), [
            'metric' => 'temperature', 'value' => 30, // no unit
        ]);
        $r->assertStatus(201);
        $reading = MachineConditionReading::query()->where('machine_id', $m->id)->first();
        $this->assertSame('celsius', $reading?->unit);
    }
}
